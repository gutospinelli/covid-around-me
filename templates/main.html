Covid ao meu redor

Olá, esse é um projeto que acessa dados de plataformas abertas como www.data.rio e www.brasil.io para mostrar quantos casos de Covid foram encontrados ao redor de você, a partir de dados anônimos e indivudializados divulgados pelas secretarias municipais de saúde.

Atualmente nossa base de dados está configurada somente para a cidade do Rio de Janeiro. Caso você possua acesso a dados individualizados de outros municípios, sinta-se a vontade para contribuir com esse projeto ou para fazer um fork dele a partir do github.
